GOAL
==============

Survive for 3 minutes.

KEYBOARD CONTROLS
==============

- W - Jump/Double Jump/Hover
- A - Left
- S - Fall faster
- D - Right
- Space - Slow motion/Invulnerability
- P or R - Pause
- F - Full Screen
- C - Quality: High -> Low -> Medium -> High
MOUSE CONTROLS
==============

- Cursor - Aim
- Left Click - Shoot
- Scroll - Change weapon
